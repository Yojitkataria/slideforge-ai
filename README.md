# SlideForge AI

AI-powered Markdown → PPTX generator using Slide Master layouts, charts, and intelligent slide planning.

---

## 🚀 Features

* Convert Markdown (.md) → PowerPoint (.pptx)
* Uses uploaded Slide Master / template
* Automatic slide planning (10–15 slides)
* Charts from numeric data
* Tables from markdown
* Infographic layouts
* Gradio UI for easy usage
* Download generated PPTX

## ⚙️ How It Works

Markdown Input
→ Parser
→ Slide Planner
→ Layout Mapping
→ Template Slide Master
→ Charts & Tables
→ PPT Builder
→ Download PPTX

---

## ▶️ Run Locally

```bash
pip install -r requirements.txt
python ui/app.py
```

Open:

```
http://127.0.0.1:7861
```

---

## 🧪 Usage

1. Upload Markdown (.md)
2. Upload Template PPT (optional)
3. Click Submit
4. Download Generated PPTX
